▸ How is available inventory calculated?
Available inventory formula:

available = on_hand − committed_units + incoming_units

Days of cover formula:

days_of_cover = available ÷ avg_daily_sales

where avg_daily_sales is derived from recent 30–90 day sales velocity. If incoming_units is missing, it is treated as 0. If committed_units is missing, no deduction is applied.
